How to Create a New Database on Oracle 8i

By Alexander Geldutes

Suppose that you have Oracle 8i & Unix
  1. Set Unix environment for the new database:
    • ORACLE_SID - name of the new database
    • ORACLE_HOME - path to the Oracle home directory
    • PATH has to include $ORACLE_HOME/bin directory

    Attention: Make sure that your environment is set correctly.
    If you made a mistake - your next actions may corrupt existing database.

  2. Create a new init.ora file.
    You may make a copy of this file from existing database or make a copy of default file from Oracle:

    $ cp $ORACLE_HOME/dbs/init.sql $ORACLE_HOME/dbs/init

  3. Update just created init.ora file, set your:
    • INSTANCE_NAME
    • DB_NAME
    • DB_BLOCK_SIZE - size in bytes of Oracle database blocks
    • CONTROL_FILES - names and location of control files of the new database
    • any others parameters that you would like to change

  4. Edit the following Korn scripts, see comments to them:

    cr_db8.sh - executes CREATE DATABASE command
    cr_db8_2.sh - creates tablespaces, rollback segments, runs catalog.sql
    cr_db8_3.sh - runs catproc.sql, caths.sql, otrcsvr.sql, pupbld.sql
  5. Run these scripts from UNIX. Check log file of the current script before running a next one.

  6. Change passwords of SYSTEM and SYS users:

    ALTER USER SYSTEM IDENTIFIED BY ... ;

    ALTER USER SYS IDENTIFIED BY ...;

  7. Update the following files: /etc/oratab, tnsnames.ora, listener.ora
