Server Clustering Nodes

What are server clustering nodes?

Server clustering nodes are configurations where arrays of non-host CPUs occupy the available chassis slots and operate as independent servers. The information from each server is collected and distributed to the IP network using embedded Ethernet switches. Server cluster nodes often act as call control nodes in a softswitch configuration.

What products do we provide in this application?

IPnexus™ CPC4400: Provides inter-connectivity between the individual servers within the chassis as well gigabit uplinks to the Internet.
