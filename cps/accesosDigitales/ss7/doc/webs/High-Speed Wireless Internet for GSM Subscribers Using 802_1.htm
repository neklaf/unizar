Application Examples

High-Speed Wireless Internet for GSM Subscribers Using 802.11

  • Seamless High-Speed Internet Service to GSM Customers
  • Ideal for Carriers or WLAN Roaming Brokers
  • Carrier-Grade Signaling Gateway
  • Billing from a Single Source
  • Potential 3G Service Alternative

What products do we provide in this application?

  • SEGway™ Signaling Gateways: Works with the Home Location Register (HLR) to validate subscriber identity and service preferences.
  • SEGway Link Replacement: Save telecom carriers the leasing or provisioning costs associated with dedicated long haul SS7 links. It can be used as a full replacement or, more practically, as an offload option to allow network planners to more effectively control network growth.
  • WAIN (Wireless Access Internet Node) Solution: Provided by Transat Technologies, it verifies subscribers identities and creates a direct connection to the Internet or GRX, then creates the billing record that's sent to the home network.
